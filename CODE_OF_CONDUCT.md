# Code of Conduct

## Our Pledge

This project follows the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Standards

We are committed to providing a welcoming and inclusive environment for all contributors.

Expected behaviors:

- Using welcoming and inclusive language
- Being respectful of differing viewpoints
- Gracefully accepting constructive feedback
- Focusing on what is best for the community
- Showing empathy towards others

## Enforcement

Project maintainers are responsible for clarifying standards and will take appropriate and fair corrective action in response to behavior violations.

## Scope

This Code of Conduct applies within all project spaces and when representing the project in public spaces.

## Contact

Report issues to the project maintainers through GitHub issues or direct contact.
